# 输出骨架（论证功能，非固定模板）

> ⚠️ 本文件只标注每段的**论证功能**，不提供固定句式。具体表达由模型根据研究内容自适应生成。
> 文献综述的结构是从NSFC成功本子中"学习出来"的，不是"写死出来"的。

---

## 一、文献综述主体的论证功能骨架

### 整体结构
```
1.2 国内外研究现状及发展动态分析
├── 1.2.1 主题1（对应研究内容1）
│   ├── 引入段
│   ├── 子主题1.1（文献群1）
│   ├── 子主题1.2（文献群2）
│   └── 主题评述段（嵌入式评述）
├── 1.2.2 主题2（对应研究内容2）
│   ├── 引入段
│   ├── 子主题2.1（文献群1）
│   ├── 子主题2.2（文献群2）
│   └── 主题评述段（嵌入式评述）
├── 1.2.3 主题3（对应研究内容3）
│   ├── 引入段
│   ├── 子主题3.1（文献群1）
│   ├── 子主题3.2（文献群2）
│   └── 主题评述段（嵌入式评述）
└── [1.2.4 或 1.2.5 文献评述]（独立评述小节，可选）
```

### 主题段的论证功能骨架

#### 引入段
- **论证功能**：界定本主题的研究范围，预告分论点
- **内容要素**：
  - 说明本主题的研究背景和重要性
  - 预告本主题将讨论的子主题
  - 引出后续文献综述

#### 子主题段（文献群）
- **论证功能**：呈现现有研究成果，构建文献群
- **内容要素**：
  - 引用具体文献，说明其研究内容
  - 按逻辑顺序组织文献（非时间顺序）
  - 文献间用连接词衔接（如、此外、然而）
  - 嵌入对比和转折，为评述铺垫

#### 主题评述段（嵌入式评述）
- **论证功能**：归纳本主题的研究进展和不足
- **内容要素**：
  - 以"综上所述"或"综上"引出
  - 总结本主题的研究进展
  - 指出本主题的研究空白
  - 为独立评述或后续研究铺垫

### 表格段（可选）
- **论证功能**：对比不同方法的优缺点
- **内容要素**：
  - 表格列：文献、方法/研究内容、评述/优势、局限
  - 表格后1段文字总结表格内容并指出不足

## 二、文献评述的论证功能骨架

### 独立评述小节结构
```
1.2.X 文献评述
├── 开头段（概括各主题的研究价值）
├── 主题评述段1（指出主题1的局限和空白）
├── 主题评述段2（指出主题2的局限和空白）
├── 主题评述段3（指出主题3的局限和空白）
└── 总结段（综合所有空白，引出本项目研究方向）
```

### 各段的论证功能

#### 开头段
- **论证功能**：概括各主题的研究价值，引出评述
- **内容要素**：
  - 概括各主题的研究进展
  - 肯定现有研究的贡献
  - 引出评述的必要性

#### 主题评述段
- **论证功能**：指出每个主题的局限和空白
- **内容要素**：
  - 指出现有研究的进展（"目前关于...的研究相对丰富"）
  - 识别研究空白和局限（"但...缺乏综合考虑...的研究"）
  - 批判性判断（方法性/数据适用性/模型解释能力/研究边界）

#### 总结段
- **论证功能**：综合所有空白，引出本项目研究方向
- **内容要素**：
  - 综合各主题的研究空白
  - 引出本项目的研究方向（"因此，本项目拟..."）
  - 说明本项目的创新点（"首先...然后...最后..."）
  - 建立"空白→创新"的论证链

## 三、嵌入式评述的论证功能骨架

### 嵌入式评述结构
```
1.2 国内外研究现状及发展动态分析
├── 1.2.1 主题1
│   ├── 引入段
│   ├── 文献综述段
│   └── 嵌入式评述段（"综上所述..."）
├── 1.2.2 主题2
│   ├── 引入段
│   ├── 文献综述段
│   └── 嵌入式评述段（"综上所述..."）
└── 1.2.3 主题3
    ├── 引入段
    ├── 文献综述段
    └── 嵌入式评述段（"综上所述..."）
```

### 嵌入式评述段
- **论证功能**：在每个主题段末承担评述功能
- **内容要素**：
  - 以"综上所述"或"综上"引出
  - 总结本主题的研究进展
  - 指出本主题的研究空白
  - 为后续研究铺垫

## 四、文献表格的输出骨架

### 类型1：方法对比表骨架

```
表X XXX方法总结/对比
┌──────────┬──────────────────┬──────────────────┐
│ 分类（可选） │ 文献              │ 方法或研究内容       │ 评述              │
├──────────┼──────────────────┼──────────────────┤
│ 方法A     │ Author等(Year)   │ 研究内容描述       │ 一句话批判性评价    │
│          │ Author等(Year)   │ 研究内容描述       │ 一句话批判性评价    │
├──────────┼──────────────────┼──────────────────┤
│ 方法B     │ Author等(Year)   │ 研究内容描述       │ 一句话批判性评价    │
│          │ Author等(Year)   │ 研究内容描述       │ 一句话批判性评价    │
└──────────┴──────────────────┴──────────────────┘
```

- **评述列**：不是摘要，是一句话批判性评价（指出贡献或局限）
- **位置**：综述→表格→评述，或引入→表格→评述
- **表格后**：1段文字总结表格内容并指出不足

### 类型2：应用汇总表骨架

```
表X 近5年XXX相关研究
┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 分类      │ 应用场景   │ 方法      │ 目标（可选）│ 评述      │ 文献      │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│ 类别1     │ 场景A     │ 方法描述   │ 目标描述   │ 简短评价   │ Author等  │
│          │ 场景B     │ 方法描述   │ 目标描述   │ 简短评价   │ Author等  │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│ 类别2     │ 场景C     │ 方法描述   │ 目标描述   │ 简短评价   │ Author等  │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘
```

- **命名策略**：常用"近5年XXX相关研究"限定时效性
- **列数**：4-6列，根据研究内容调整
- **表格后**：1段文字总结表格内容并指出不足

### 类型3：特征矩阵表骨架

```
表X XXX研究特征对比
┌──────────────────┬──────────┬──────────┬──────────┬──────────┐
│ 文献              │ 特征维度1 │ 特征维度2 │ 特征维度3 │ 特征维度4 │
├──────────────────┼──────────┼──────────┼──────────┼──────────┤
│ Author等(Year)   │    √     │    √     │    ×     │    √     │
│ Author等(Year)   │    √     │    ×     │    √     │    ×     │
│ Author等(Year)   │    ×     │    √     │    √     │    √     │
└──────────────────┴──────────┴──────────┴──────────┴──────────┘
```

- **标记方式**：√/× 标记覆盖情况
- **功能**：空白维度即研究空白
- **表格后**：1段文字指出缺失维度和研究空白

## 五、参考文献的输出骨架

### BibTeX格式引用库
```bibtex
@article{key1,
  title = {文献标题},
  author = {作者1 and 作者2 and 作者3},
  journal = {期刊名称},
  year = {年份},
  volume = {卷},
  number = {期},
  pages = {页码},
  doi = {DOI},
  abstract = {摘要（可选）}
}

@inproceedings{key2,
  title = {会议论文标题},
  author = {作者1 and 作者2},
  booktitle = {会议名称},
  year = {年份},
  pages = {页码},
  doi = {DOI}
}

@book{key3,
  title = {书籍标题},
  author = {作者},
  publisher = {出版社},
  year = {年份},
  isbn = {ISBN}
}
```

### 正文引用格式
- **作者-年份制**：`作者等(年份)` 或 `Author et al.(年份)`
- **编号制**：`[1]`、`[2]`、`[3]`

## 六、骨架使用说明

1. **骨架是论证功能的标注，不是固定模板**
2. **具体表达由模型根据研究内容自适应生成**
3. **主题数量、子主题数量根据项目类型调整**
4. **评述方式（嵌入式/独立式）根据项目类型选择**
5. **表格使用根据文献数量和方法类型判断**
6. **引用格式根据用户指定选择**
