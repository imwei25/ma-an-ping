# NSFC 文献综述组织逻辑范式

> 本文件从16本NSFC成功申请书中归纳不同研究类型的文献组织差异、方法型/机制型/系统型文献组织差异、文献如何支撑NSFC论证结构。
> **核心原则**：范式是从本子中归纳的"组织模式"，不是"固定模板"。模型应根据用户研究内容自适应选择和调整。

---

## 一、五种项目类型的典型文献综述组织

### 1. 机制研究型
- **组织方式**：按研究内容分主题，理论→机理→实证
- **文献评述方式**：独立评述，指出理论空白
- **典型结构**：
  - 主题1：理论基础（理论A、理论B的演进）
  - 主题2：机理研究（机制A、机制B的对比）
  - 主题3：实证研究（实证发现、实证不足）
  - 主题4：前沿方向（新兴理论、未解问题）
- **文献群构建**：理论文献群→机理文献群→实证文献群
- **评述重点**：理论解释力不足、机理研究不深入、实证证据不充分

### 2. 数据驱动型
- **组织方式**：按研究内容分主题，数据→特征→模型→验证
- **文献评述方式**：独立评述，指出方法空白
- **典型结构**：
  - 主题1：数据来源与预处理
  - 主题2：特征工程方法
  - 主题3：建模方法（传统方法→深度学习方法）
  - 主题4：验证与评估方法
- **文献群构建**：按方法类型构建文献群，同类方法聚集
- **评述重点**：数据规模不足、特征提取不充分、模型泛化能力弱、评估指标单一

### 3. 方法研究型
- **组织方式**：按理论/方法/应用三方面分主题，含大量表格
- **文献评述方式**：独立评述，指出方法局限
- **典型结构**：
  - 主题1：理论基础（理论框架演进）
  - 主题2：方法演进（方法A→方法B→方法C）
  - 主题3：应用研究（应用场景拓展）
  - 主题4：方法对比（含表格）
- **文献群构建**：按方法类别构建，表格对比不同方法
- **评述重点**：方法假设过强、适用范围有限、计算复杂度高、可解释性不足

### 4. 实证研究型
- **组织方式**：按研究内容分主题，文献→数据→实证
- **文献评述方式**：嵌入式评述，指出实证空白
- **典型结构**：
  - 主题1：研究问题界定
  - 主题2：数据来源与样本
  - 主题3：实证方法与发现
  - 主题4：实证不足与改进
- **文献群构建**：按实证发现构建，对比不同研究的结论
- **评述重点**：样本代表性不足、变量测量不精确、因果识别不严格、外部效度有限

### 5. 优化决策型
- **组织方式**：按研究内容分主题，问题→方法→优化
- **文献评述方式**：综合评述段，指出优化空间
- **典型结构**：
  - 主题1：问题界定与测量
  - 主题2：评估方法
  - 主题3：优化模型
  - 主题4：决策应用
- **文献群构建**：按优化目标构建，对比不同优化策略
- **评述重点**：目标函数简化、约束条件不全、求解效率低、决策支持不足

## 二、方法型/机制型/系统型文献组织差异

### 方法型文献组织模式
- **核心特征**：按方法类型分类，含表格对比
- **组织逻辑**：
  1. 方法类别划分（如传统方法/机器学习方法/深度学习方法）
  2. 每类方法下按演进顺序组织
  3. 方法间用表格对比优缺点
  4. 评述指出方法局限
- **表格使用**：高频（方法对比表、性能对比表）

### 机制型文献组织模式
- **核心特征**：按研究内容分类，理论→机理→实证
- **组织逻辑**：
  1. 理论文献群（理论基础演进）
  2. 机理文献群（机制研究进展）
  3. 实证文献群（实证发现与不足）
  4. 评述指出理论空白
- **表格使用**：低频（主要用于实证发现对比）

### 系统型文献组织模式
- **核心特征**：按系统模块分类，需求→架构→模块→集成
- **组织逻辑**：
  1. 需求分析文献群
  2. 系统架构文献群
  3. 模块设计文献群
  4. 系统集成文献群
  5. 评述指出系统空白
- **表格使用**：中频（主要用于模块功能对比）

## 三、文献群的构建逻辑

### 文献群构建原则
1. **同主题聚集**：同一研究主题的文献放在一起
2. **对比关系构建**：方法/观点不同的文献构成对比关系
3. **递进关系构建**：理论→方法→应用的文献构成递进关系
4. **互补关系构建**：从不同角度研究同一问题的文献构成互补关系

### 文献群构建模式（从16本归纳）

#### 模式1：方法对比群
- **构建方式**：将不同方法的文献放在一起对比
- **适用场景**：方法研究型、数据驱动型
- **示例**：传统方法文献 + 机器学习方法文献 + 深度学习方法文献

#### 模式2：视角互补群
- **构建方式**：将不同视角的文献放在一起互补
- **适用场景**：机制研究型、实证研究型
- **示例**：理论视角文献 + 实证视角文献 + 应用视角文献

#### 模式3：演进递进群
- **构建方式**：将同一研究方向的文献按演进顺序组织
- **适用场景**：所有类型
- **示例**：早期研究文献 + 中期改进文献 + 最新进展文献

#### 模式4：问题聚焦群
- **构建方式**：将研究同一问题的文献放在一起
- **适用场景**：优化决策型、实证研究型
- **示例**：问题界定文献 + 测量方法文献 + 优化策略文献

## 四、文献→研究问题映射模式

### 映射逻辑
1. **研究内容→文献主题**：每个研究内容对应一个文献主题
2. **文献主题→研究空白**：每个文献主题的评述指出研究空白
3. **研究空白→研究问题**：研究空白直接对应本项目的研究问题

### 映射模式（从16本归纳）

#### 模式1：直接对应
- 文献主题1的空白 → 研究问题1
- 文献主题2的空白 → 研究问题2
- 文献主题3的空白 → 研究问题3

#### 模式2：综合对应
- 多个文献主题的空白综合 → 关键科学问题

#### 模式3：递进对应
- 文献主题1的空白 → 研究问题1（基础）
- 文献主题2的空白 → 研究问题2（进阶）
- 文献主题3的空白 → 研究问题3（应用）

---

## 五、文献→方法合理性映射模式

### 映射逻辑
1. **文献综述介绍方法**：呈现现有各种方法
2. **指出现有方法局限**：评述指出方法不足
3. **本项目方法改进**：本项目方法是对现有方法的改进或创新
4. **技术路线衔接**：技术路线部分引用"如综述部分所见"

### 映射模式（从16本归纳）

#### 模式1：方法改进型
- 现有方法A有局限X → 本项目方法A'改进了局限X

#### 模式2：方法融合型
- 现有方法A有局限X，现有方法B有局限Y → 本项目融合A和B，克服X和Y

#### 模式3：方法创新型
- 现有方法均无法解决某问题 → 本项目提出全新方法

---

## 六、文献→创新合理性映射模式

### 映射逻辑
1. **文献评述指出研究空白**：综合各主题的空白
2. **本项目创新点对应空白**：每个创新点对应一个空白
3. **形成"空白→创新"论证链**：创新点不是凭空产生，而是填补空白

### 映射模式（从16本归纳）

#### 模式1：空白填补型
- 空白X → 创新点1填补空白X
- 空白Y → 创新点2填补空白Y

#### 模式2：方法突破型
- 现有方法局限 → 创新点突破方法局限

#### 模式3：视角拓展型
- 现有研究视角单一 → 创新点拓展研究视角

---

## 七、文献评述的4种批判性判断类型

### 1. 方法性评价
- **判断对象**：现有研究的方法
- **判断内容**：方法的局限、假设的强度、适用范围
- **典型表达**：指出现有方法"多关注于X，对Y处理效率较低"

### 2. 数据适用性评价
- **判断对象**：现有研究的数据
- **判断内容**：数据规模、数据质量、数据来源
- **典型表达**：指出现有研究"对大量不确定信息涉及很少"

### 3. 模型/机制解释能力评价
- **判断对象**：现有研究的模型/机制
- **判断内容**：模型的解释能力、机制的揭示程度
- **典型表达**：指出"模型黑盒性难以解释推荐结果"

### 4. 研究边界与局限识别
- **判断对象**：现有研究的范围
- **判断内容**：研究边界、研究深度、研究广度
- **典型表达**：指出"现有研究大多集中于某个因素，缺乏综合考虑"

### 批判性判断的组织方式
- **嵌入式**：每个主题段末的"综上所述"承担批判性判断
- **独立式**：独立评述小节综合各主题的批判性判断
- **混合式**：嵌入式 + 独立式结合

## 八、文献组织的选择决策树

```
用户研究构想
    │
    ├─ 研究内容明确且独立？─是→ 按研究内容划分主题（策略1）
    │
    ├─ 方法分类清晰？─是→ 按方法类型划分主题（策略2）
    │
    ├─ 研究视角多元？─是→ 按研究视角划分主题（策略3）
    │
    └─ 检测/对象明确分类？─是→ 按检测对象划分主题（策略4）
```

```
项目类型判断
    │
    ├─ 机制研究型？→ 理论→机理→实证组织 + 独立评述
    ├─ 数据驱动型？→ 数据→特征→模型→验证组织 + 独立评述
    ├─ 方法研究型？→ 理论/方法/应用组织 + 表格对比 + 独立评述
    ├─ 实证研究型？→ 文献→数据→实证组织 + 嵌入式评述
    └─ 优化决策型？→ 问题→方法→优化组织 + 综合评述段
```

### 表格使用决策树

```
是否需要文献表格？
    │
    ├─ 项目类型判断
    │   ├─ 方法研究型？→ 几乎必用，每子主题1张
    │   ├─ 数据驱动型？→ 常用，方法对比处用
    │   ├─ 机制研究型？→ 可选，概念框架处可用
    │   ├─ 实证研究型？→ 通常不用
    │   └─ 优化决策型？→ 通常不用
    │
    ├─ 文献数量判断
    │   ├─ 子主题内文献≥5篇且方法≥3种？→ 用方法对比表
    │   ├─ 需展示方法在不同场景的应用？→ 用应用汇总表
    │   └─ 需对比研究覆盖的维度？→ 用特征矩阵表
    │
    └─ 表格类型选择
        ├─ 对比不同方法的优劣？→ 方法对比表（文献/方法/评述）
        ├─ 展示方法的应用广度？→ 应用汇总表（分类/场景/方法/评述/文献）
        └─ 对比研究覆盖维度？→ 特征矩阵表（文献/维度列/√×标记）
```
