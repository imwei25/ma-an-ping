# 术语表

## 研究方法相关

| 术语 | 英文 | 说明 |
|---|---|---|
| 文献研究法 | Literature Review | 系统梳理和分析已有文献 |
| 数据挖掘 | Data Mining | 从大量数据中发现隐含模式 |
| 机器学习 | Machine Learning | 让计算机从数据中学习规律 |
| 深度学习 | Deep Learning | 基于多层神经网络的学习方法 |
| 数学建模 | Mathematical Modeling | 用数学语言描述实际问题 |
| 优化方法 | Optimization Method | 寻找最优解的方法 |
| 多准则决策 | Multi-Criteria Decision Making | 在多个准则下进行决策 |
| 实证研究 | Empirical Research | 基于实际数据的验证研究 |
| 扎根理论 | Grounded Theory | 从数据中自下而上构建理论 |
| 系统动力学 | System Dynamics | 研究系统动态行为的方法 |
| 实施科学 | Implementation Science | 研究如何将证据转化为实践 |
| 德尔菲法 | Delphi Method | 通过多轮专家咨询达成共识 |
| 因果推断 | Causal Inference | 识别因果关系的统计方法 |
| 文本挖掘 | Text Mining | 从文本数据中提取有用信息 |
| 关联挖掘 | Association Mining | 发现数据项之间的关联关系 |
| 元分析 | Meta-Analysis | 对多个研究结果进行统计综合 |
| 群体画像 | User Profiling | 基于数据构建用户特征画像 |
| 集成学习 | Ensemble Learning | 组合多个学习器提升性能 |
| 异常检测 | Anomaly Detection | 识别与正常模式不同的数据 |
| 社区发现 | Community Detection | 在网络中发现密集连接的子图 |
| 特征选择 | Feature Selection | 选择最相关的特征子集 |
| 特征融合 | Feature Fusion | 将多种特征进行整合 |
| 强化学习 | Reinforcement Learning | 通过与环境交互学习策略 |
| 迁移学习 | Transfer Learning | 将已有知识迁移到新任务 |
| 对比学习 | Contrastive Learning | 通过对比样本学习表征 |

## 技术路线相关

| 术语 | 英文 | 说明 |
|---|---|---|
| 技术路线 | Technical Route | 研究的总体技术路径 |
| 技术路线图 | Technical Roadmap | 技术路线的可视化表示 |
| 研究方案 | Research Scheme | 研究的具体实施方案 |
| 可行性分析 | Feasibility Analysis | 论证研究方案的可执行性 |
| 闭环 | Closed Loop | 包含反馈和验证的完整流程 |
| 输入输出 | Input-Output | 模块的输入和输出关系 |
| 模块 | Module | 技术路线中的功能单元 |

## 创新特色相关

| 术语 | 英文 | 说明 |
|---|---|---|
| 特色 | Feature/Characteristic | 研究在范式/视角层面的独特性 |
| 创新 | Innovation | 研究在科学问题上的突破点 |
| 方法创新 | Methodological Innovation | 提出新方法或改进现有方法 |
| 视角创新 | Perspective Innovation | 从新视角研究问题 |
| 理论创新 | Theoretical Innovation | 提出新理论或新范式 |
| 应用创新 | Application Innovation | 在新领域应用已有方法 |
| 框架创新 | Framework Innovation | 构建新的分析框架 |
| 集成创新 | Integrative Innovation | 多方法协同创新 |
| 关键科学问题 | Key Scientific Problem | 研究的核心科学难点 |
