# 质量验证菜单

> 写作完成后，必须逐项检查以下清单。

---

## A. 闭环检查

- [ ] 每个研究目标有对应的专题（1:1映射）
- [ ] 关键科学问题覆盖所有专题
- [ ] 专题之间有明确递进关系
- [ ] 研究目标→研究内容→关键科学问题形成完整闭环

## B. 研究内容结构检查

- [ ] 总起大帽段：180-250字
- [ ] 3-4个专题
- [ ] 每个专题有专题帽段：100-200字
- [ ] 每个专题有2-4个孙研究内容（一般为3个）
- [ ] 专题间有明确递进关系

## C. 6步语义顺序检查（每个孙研究内容段落）

- [ ] 复杂问题引入句：从宏观系统角度描述复杂性
- [ ] 现实运行困境句：指出关键管理困境
- [ ] 现有研究局限句：指出现有研究不足/GAP
- [ ] 本质科学矛盾句：上升为更高阶理论矛盾
- [ ] 理论突破需求句：提出需要构建新型方式
- [ ] 科学问题落点句：体现最终目标指向和科学贡献

## D. 总起大帽段检查

- [ ] 字数180-250字
- [ ] 概括3-4个专题的标题和逻辑关系
- [ ] 强调科学问题复杂性
- [ ] 引出结构性矛盾/治理瓶颈
- [ ] 点出knowledge gap

## E. 专题帽段检查

- [ ] 字数100-200字
- [ ] 说明本专题的研究思路和内部逻辑
- [ ] 概括孙研究内容之间的关系

## F. 孙研究内容段落字数检查

- [ ] 每段300-400字

## G. 反模式检查

- [ ] 无绝对化表述（"首次""领先""填补空白""突破性""国际领先"）
- [ ] 无论文体表达（"本文""本文认为"）
- [ ] 无技术实现细节过度下沉
- [ ] 无简单并列的任务列表（专题间必须有递进关系）
- [ ] 无孤立的研究内容
- [ ] 无1:1机械映射（科学问题≠专题标题的简单重复）

## H. 一致性检查

- [ ] 术语跨章节一致
- [ ] 研究对象/缩写/指标口径一致
- [ ] 目标-内容-问题形成完整映射
- [ ] 首次出现给出缩写与定义

## I. 递进逻辑验证

- [ ] 删除任一专题，其余内容是否还能成立？
  - 如果能 → 递进关系不够紧密
  - 如果不能 → 递进关系成立
- [ ] 前一个专题的输出是否是后一个专题的输入？

## J. 字数检查

| 章节 | 面上项目 | 重点项目 | 青年项目 | 省基金 |
|---|---|---|---|---|
| 研究目标 | 300-500字 | 500-800字 | 200-400字 | 300-500字 |
| 研究内容 | 2000-3000字 | 4000-6000字 | 1500-2500字 | 2000-3000字 |
| 关键科学问题 | 500-800字 | 800-1500字 | 400-600字 | 500-800字 |

### 数量检查
