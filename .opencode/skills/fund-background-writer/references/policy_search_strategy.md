# 政策检索策略

> 用于立项依据研究背景部分的政策引用，确保开篇有国家/部门级政策支撑。

## 检索关键词策略

### 通用关键词模板

```
[研究领域] + [政策类型关键词]
```

政策类型关键词：
- 规划 / 纲要 / 战略 / 意见 / 方案 / 行动计划 / 通知 / 指导意见

### 示例

| 研究领域 | 检索关键词 |
|----------|-----------|
| 医疗AI | "医疗人工智能 政策" "智慧医院 指导意见" "健康中国2030" |
| 医保欺诈 | "医疗保障基金监管 条例" "医保基金 欺诈 政策" |
| 数据安全 | "数据安全法" "个人信息保护法" "网络安全 审查" |
| 双碳 | "碳达峰 碳中和 意见" "绿色低碳 发展规划" |

## 政策引用格式

在立项依据中引用政策时，必须包含以下信息：

```
《[政策文件全称]》（[发布年份]）[核心条款/要求]
```

示例：
- 《"健康中国2030"规划纲要》（2016）明确提出要强化慢性病筛查和综合干预
- 《医疗保障基金使用监督管理条例》（2021）要求加强医保基金监管

---

## 引用数量要求

- 研究意义部分至少引用 2-3 个国家/部门级政策文件
- 优先使用近 5 年内发布的政策
- 如有国际政策对比需求，可补充 1 个国际政策

---

## 注意事项

1. **政策必须真实存在**：不可编造政策文件名或条款内容
2. **政策必须可追溯**：记录政策来源URL或官方发布文号
3. **优先使用最新政策**：同一主题有多个政策时，优先使用最新版本
4. **政策与研究主题必须相关**：不可生硬套用无关政策
