# 数据检索策略

> 用于立项依据研究背景部分的统计数据引用，确保开篇有权威数据支撑问题严重性。

---

## 优先来源（按优先级排序）

### 第一梯队：国家级统计数据

| 来源 | 适用领域 | 典型数据 |
|------|----------|----------|
| 国家统计局 | 全领域 | GDP、人口、行业产值、就业数据 |
| 中国卫生健康统计年鉴 | 医疗卫生 | 发病率、死亡率、医疗资源、医保支出 |
| 中国教育统计年鉴 | 教育科技 | 在校生数、教育经费、师资比 |
| 中国科技统计年鉴 | 科技创新 | R&D投入、专利数、论文数 |

### 第二梯队：行业权威报告

| 来源 | 适用领域 | 典型数据 |
|------|----------|----------|
| 行业蓝皮书 | 各行业 | 行业规模、增长率、市场结构 |
| 权威白皮书 | 各领域 | 技术发展现状、应用规模、趋势预测 |
| 中国社科院报告 | 社会经济 | 社会发展指标、经济预测 |
| 行业协会年度报告 | 各行业 | 行业统计、会员数据、发展指标 |

### 第三梯队：国际组织报告

| 来源 | 适用领域 | 典型数据 |
|------|----------|----------|
| WHO | 全球卫生 | 疾病负担、医疗资源、卫生支出 |
| 世界银行 | 发展指标 | GDP、贫困率、发展指数 |
| OECD | 发达经济体 | 经济指标、社会指标 |
| UNDP | 人类发展 | HDI、不平等指标 |

### 第四梯队：高水平学术文献

| 来源 | 适用领域 | 典型数据 |
|------|----------|----------|
| Nature / Science | 全领域 | 全球性统计数据、meta分析结果 |
| Lancet / NEJM | 医学 | 疾病负担、临床试验数据 |
| PNAS | 多学科 | 研究趋势数据、综述数据 |
| 高水平综述论文 | 各领域 | 领域现状数据、发展指标 |

---

## 禁止来源

以下来源**严禁使用**：

1. **营销文章**：软文、广告、推广文章中的数据
2. **无来源数据**：无法追溯到原始出处的数据
3. **转载数据**：二次转引且无法确认原始来源的数据
4. **自媒体数据**：微信公众号、知乎回答等非权威来源
5. **过时数据**：超过 10 年的统计数据（除非用于趋势对比）

---

## 数据记录格式

每条使用的数据必须记录以下信息：

```
- 数据项：[描述统计什么]
- 年份：[数据年份]
- 具体数值：[数字+单位]
- 出处：[来源名称+具体文献/报告名]
- 原始链接：[URL]
```

### 示例

```
- 数据项：中国慢性病导致的死亡占总死亡比例
- 年份：2023
- 具体数值：88.5%
- 出处：中国卫生健康统计年鉴2024
- 原始链接：http://www.nhc.gov.cn/...
```

---

## 引用数量要求

- 研究意义部分至少使用 3-5 个统计数据
- 研究背景前 2 段必须有具体统计数据
- 优先使用近 3 年内的数据
- 如有趋势对比需求，可补充历史数据

---

## 数据在立项依据中的使用方式

### 数据→问题模式

```
根据[来源]数据显示，[具体数值]。由于[原因]，导致[问题]。
```

### 趋势→紧迫性模式

```
[来源]数据显示，[指标]从[年份]的[数值1]增长至[年份]的[数值2]，
[增长/变化幅度]，[问题]日益严峻。
```

### 对比→差距模式

```
我国[指标]为[数值]，而[发达国家/全球平均]为[数值]，
差距显著，[问题]亟待解决。
```

---

## 注意事项

1. **数据必须真实**：不可编造数据
2. **数据必须可追溯**：每条数据必须有出处和原始链接
3. **数据必须是最新的**：优先使用最新年份数据
4. **数据必须与研究主题相关**：不可生硬套用无关数据
5. **数据单位必须一致**：同一指标在不同年份的数据单位必须一致
6. **注意数据口径**：不同来源的同类数据可能口径不同，不可直接比较
