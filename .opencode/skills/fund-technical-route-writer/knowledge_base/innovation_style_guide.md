# 创新和特色写作风格指南

> 基于NSFC成功申请书和规范的逻辑提炼，禁止迁移具体研究内容。

## 一、创新和特色的结构规范

### 标准结构（面上/青年/省基金）

```
4．本项目的特色与创新之处

（1）[创新点1]
（2）[创新点2]
（3）[创新点3]
```

### 重点项目结构

```
4．本项目的特色与创新之处

4.1 本项目的特色之处
（1）[特色1]
（2）[特色2]
（3）[特色3]

4.2 本项目的创新之处
（1）[创新1]（对应关键科学问题一）
（2）[创新2]（对应关键科学问题二）
（3）[创新3]（对应关键科学问题三）
（4）[创新4]（对应关键科学问题四）
（5）[创新5]（对应关键科学问题五）
```

**关键区别**：重点项目将"特色"和"创新"分开写——
- 特色强调研究范式、视角和方法论层面的独特性
- 创新强调具体科学问题的突破点

## 二、创新点表述结构

### 模式A：简洁型（10/16本采用）

```
（N）[一句话概括创新点]。[突破什么局限/解决什么问题]。[本项目如何实现]。
```

示例结构：
```
（1）提出[方法/框架/视角]。针对现有研究[不足/局限]，本项目[如何突破]，
[实现什么价值]。
```

适用：面上项目、青年项目、省基金

### 模式B：论证型（6/16本采用）

```
（N）[标题句：一句话概括创新点]

针对[科学难题/方法难题/策略难题]（对应关键科学问题X），本项目突破
[传统方法的局限]，[本项目的创新做法]，[实现什么跨越/跃升]。
```

适用：重点项目、需要强化创新论证的项目

### 表述要素优先级

1. **必须**：创新点概括 + 突破什么局限
2. **推荐**：与现有方法的差异 + 价值贡献
3. **可选**：对应关键科学问题（重点项目必须）

## 三、创新和特色与研究内容的映射规范

### 核心原则

- 每个创新点必须能在研究内容中找到落点
- 创新点与关键科学问题呼应
- 创新点层次高于具体研究内容

### 映射方式

**方式1：隐式映射**（面上/青年/省基金）
- 创新点与研究内容对应，但不显式标注
- 读者可从内容推断对应关系

**方式2：显式映射**（重点项目推荐）
- 每个创新点明确标注"对应关键科学问题X"
- 形成严格的一一对应

### 映射数量规范

- 面上项目：3-4个创新点，对应3-4个研究内容
- 重点项目：5个创新点，对应5个课题/关键科学问题
- 青年项目：3-4个创新点，对应2-3个研究内容
- 省基金：3-4个创新点，对应3个研究内容

## 四、创新和特色与研究方法的映射规范

### 映射逻辑

创新点通常同时对应研究内容和研究方法：
```
创新点 = 研究内容的突破 + 研究方法的改进
```

### 表述方式

在创新点中体现方法创新：
```
提出[方法名称]，突破[现有方法的局限]，实现[研究目标]。
```

## 五、创新类型及表述规范

| 创新类型 | 标准表述模板 | 出现频率 |
|---|---|---|
| 方法创新 | "提出[方法]，解决[问题]" | 100% |
| 视角创新 | "从[视角]出发，不同于以往[范式]" | 50% |
| 理论创新 | "提出[理论/范式]，突破传统[局限]" | 44% |
| 应用创新 | "将[方法]应用于[领域]，实现[目标]" | 38% |
| 框架创新 | "构建[框架/体系]，实现[功能]" | 31% |
| 集成创新 | "融合[A]与[B]，突破[瓶颈]" | 25% |

## 六、字数规范

| 项目类型 | 创新特色字数 | 创新点数量 |
|---|---|---|
| 面上项目 | 500-800字 | 3-4点 |
| 重点项目 | 800-1500字 | 特色3点+创新5点 |
| 青年项目 | 400-600字 | 3-4点 |
| 省基金 | 400-700字 | 3-4点 |

## 七、写作禁忌

1. **避免绝对化表述**：慎用"首次""领先""填补空白"（但非绝对禁止，需有依据）
2. **避免创新点与研究内容脱节**：每个创新点必须能在研究内容中找到落点
3. **避免创新点只是口号**：必须有具体论证
4. **避免堆砌创新点**：3-4个充分论证的创新点优于5个薄弱的
5. **避免创新点之间逻辑重叠**：每个创新点应有独立的论证空间
6. **避免只说"做了什么"**：必须说"为什么是创新"

## 七点五、排版提醒

1. **创新点标题句加粗**：每个创新点的概括句加粗，方便评审专家快速定位
2. **使用编号列表**：（1）（2）（3）清晰编号，不要用无序列表
3. **关键词突出**：方法名称、理论名称等关键术语可适当突出
4. **段落简洁**：每个创新点控制在1段内，避免大段文字墙

## 八、规范要点

1. 创新点应与关键科学问题相呼应
2. 主要写清楚"为什么是创新"
3. 呼应国内外研究现状分析中的总结结论
4. 强调前人没有研究过或研究不深
5. 如果有研究视角/观点/思路的不同 → "有特色"
6. 如果有新的理论方法或在新领域的创新应用 → "有创新"
7. 创新点2-3个为宜（但实际成功本子多为3-4个，适度增加可接受）
